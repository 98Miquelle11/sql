# Theory
## Queries
### SELECT FROM ⟺ select from the database
Information (simple or complex) search query.

*Example:*  login script - **"Search the user database for the login and password provided in the form."**

### INSERT INTO ⟺ add to database
Query adding a new record to the database.

*Example:*  registration script - **"Registration form that allows you to create an account in our web application."**

### UPDATE ⟺ update the value
Query replacing an existing value in the database.

*Example:*  profile editing - **"A logged-in user edits their own account settings."**

## Additional information
### AND/OR conjunction or comma?
* Before `WHERE` we insert **commas**,
* After `WHERE` we insert conjunctions: **AND**, **OR**.

### Flexible SQL language
* Ability to insert incomplete records,
* Ability to change the order of inserted data.
